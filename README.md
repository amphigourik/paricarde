# Paricarde 🚴‍♂️🥖

La parodie ultime de course cycliste, entre Paris et le saucisson.

## 🚀 Installation

1. Cloner le repo :
   ```sh
   git clone https://github.com/ton-utilisateur/paricarde.git
   cd paricarde
   ```


2. Installer les dépendances :
   ```sh
   poetry install
   ```

3. Lancer le serveur :
    ```sh
    poetry run paricarde
    ```
